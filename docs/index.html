Quick Start

1. Install uv (Python package manager)

# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Or with pip
pip install uv

2. Clone and install the framework

git clone https://github.com/pierrehanne/agent-workflow.git
cd agent-workflow

uv venv
source .venv/bin/activate
uv pip install -e ".[dev]"

3. Set up your API key

export GEMINI_API_KEY="your_api_key_here"

4. Run your first workflow

python main.py

Workflow Patterns

üîó Prompt Chaining

Execute prompts sequentially, passing outputs to inputs

from framework.chaining import ChainWorkflow

chain = ChainWorkflow(name="my_chain")
chain.add_node(step1).add_node(step2).add_node(step3)
result = chain.execute(context)

üîÄ Conditional Routing

Route execution based on conditions

from framework.routing import RouterNode, KeywordCondition

router = RouterNode(
    name="classifier",
    routes={
        "technical": (KeywordCondition(["API", "code"]), technical_node),
        "general": (KeywordCondition(["explain"]), general_node)
    }
)

‚ö° Parallel Execution

Run multiple operations concurrently

from framework.parallel import ParallelWorkflow

parallel = ParallelWorkflow(name="concurrent_tasks")
parallel.add_node(task1).add_node(task2).add_node(task3)
results = parallel.execute(context)

üéØ Orchestrator Pattern

Coordinate multiple worker agents

from framework.orchestrator import OrchestratorNode

orchestrator = OrchestratorNode(
    name="coordinator",
    workers=[worker1, worker2, worker3],
    llm_provider=provider
)

‚úÖ Evaluator-Optimizer

Evaluate and optimize outputs iteratively

from framework.evaluator import EvaluatorNode, CompositeEvaluator

evaluator = EvaluatorNode(
    target_node=summary_node,
    evaluator=CompositeEvaluator([...]),
    threshold=0.7,
    max_iterations=3
)
